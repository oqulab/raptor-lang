Raptorian Lang

Raptorian is a unique multilingual, interpreted, high-level object-oriented programming language designed for educational purposes to lower the barrier to entry for beginners, with an emphasis on intuitive, clear, and modular programming. The language supports key features of modern programming languages, including strong typing, classes, functions, and convenient control of program flow.

Get started with Raptorian


Hello world!

fun main() {
        println("Hello world!")
}


In Raptorian

  • "fun" is used to declare a function
  • the "main()" function is where your program starts from
  • the body of a function is written within curly braces  "{}"

Variables

All programs need to be able to store data, and variables help you to do just that. In Raptorian, you can declare:

  • read-only variables with "val"
  • mutable variables with "var"

To assign a value, use the assignment operator "=".

val x = 11 // There are 11 "x"
val y = 22 // There are 22 "y"
var a = x * y // Multiplication
println(a) // 242

a = 1111
println(a) // 1111

As "a" is a mutable variable, its value can be reassigned after declaration.

Basic types

Every variable and data structure in Raptorian has a data type. Data types are important because they tell the compiler what you are allowed to do with that variable or data structure. In other words, what functions and properties it has.

val x: Int = 11
val text: String = "Hello" 
val pi: Double = 3.14 
val isActive: Boolean = false
val symbol: Char = 'A' 

You can specify the type explicitly or omit specifying the type of the variable.

a = x + y // Addition
println(a) // 33

a = y / x // Division
println(a) // 2

a = x - y // Subtraction
println(a) // -11

Collections

Collections are data structures that group multiple elements, like arrays, lists, and sets. They store and manage groups of data efficiently.

val array = [1, 2, 3, 4]

val array2: List<Int> = [1, 2, 3, 4] // same as 'array' 

Methods:

size: Returns the number of elements in the array.

array.size // result will be: 4

indexOf(element): Returns the index of the first occurrence of the specified element in the array, or -1 if the element is not present.

array.indexOf(3) // result will be: 2

remove(element): Removes the first occurrence of the specified element from the array. If the element is not found, the array remains unchanged.

array.remove(1) // [2, 3, 4]

removeAt(index): Removes the element at the specified index in the array. If the index is out of range, it throws an error or does nothing.

array.removeAt(2) // [2, 3]

add(element): Appends the specified element to the end of the array.

array.add(12) // [2, 3, 12]

add(index, element): Inserts the specified element at the specified position in the array. Shifts the element currently at that position (if any) and any subsequent elements to the right.

array.add(0, 33) // [33, 2, 3, 12]

isEmpty(): Returns true if the array is empty, false otherwise.

array.isEmpty() // false

isNotEmpty(): Returns true if the array contains one or more elements, false if it is empty.

array.isNotEmpty() // true

contains(element): Checks if the array contains the specified element, returning true if it does and false otherwise.

array.contains(12) // true
array.contains(111) // false

toString(): Converts the array into a string representation.

array.toString() // “[33, 2, 3, 12]”

joinToString(separator: String = ", "): Converts and concatenates the array elements into a single string, separated by the specified string (defaults to a comma and a space).

array.joinToString(“.”) // “33.2.3.12”

reversed(): Returns a list with elements in reversed order.

array.reversed() // [12, 3, 2, 33]

Maps

Maps are a collection type that stores data as key-value pairs, allowing for efficient data retrieval and manipulation based on keys.

val map = { "a": 1, "b": 2, "c": 3 }

keys: Returns a list of all keys present in the map.

Functions

In programming, "fun" refers to a function, a block of code designed to perform a specific task. When called, a function executes its code and can return a value. Functions can take parameters, allowing for more versatile and reusable code. They are fundamental in organizing and structuring programs, making code more modular and maintainable.

Classes

A "class" in programming is a blueprint for creating objects. It defines properties (attributes) and behaviors (methods) that the objects created from the class will have. Classes encapsulate data for the object and methods to manipulate that data, promoting organized, modular, and reusable code. They are fundamental to object-oriented programming, allowing for the creation of complex, yet manageable, systems.

   var a: Int,
   val type: String,
   var active: Boolean = false,
   var level: Double = 0.0

                println("a=$a, type=$type, active=$active, level=$level")

        atom.display()

String interpolation

String interpolation is a programming technique for embedding expressions within string literals. This allows for the dynamic insertion of values, such as variables or computations, directly into strings. It simplifies the process of concatenating strings with other data types, making code more readable and concise. Interpolation is supported in many programming languages, often with a specific syntax like ${expression} or %{expression}.

println("a + 2 = ${ a + 2 }") // will be: a + 2 = 14

Characters (Char)

This section covers operations and methods available for working with single characters (Char). These operations mimic Kotlin's behavior for character arithmetic, comparisons, and type checking.

Methods:

code: Returns the ASCII/Unicode integer value of the character.

val ch = 'A'
val ascii = ch.code // result will be: 65

Increment and Decrement:

Prefix Increment / Decrement (++Char, --Char)

Increments or decrements the character's code by 1, then returns the new character.

val ch = 'a'
println(++ch) // result will be: 'b'
println(--ch) // result will be: 'a'

Postfix Increment / Decrement (Char++, Char--)

Increments or decrements the character's code by 1, then returns the new character.

val ch = 'a'
println(ch++) // result will be: 'a' (ch becomes 'b' after)
println(ch--) // result will be: 'b' (ch becomes 'a' after)

Arithmetic Operations

Concatenation (Char + String, String + Char)

Combines a character and a string into a new concatenated string.

val result1 = 'A' + "BC" // result will be: "ABC"

val result2 = "AB" + 'C' // result will be: "ABC"

Character and Integer Arithmetic (Char + Int, Char - Int)

Adds or subtracts an integer to/from the character's code, returning a new character.

val nextCh = 'a' + 2 // result will be: 'c'
val nextCh = 'c' - 2 // result will be: 'a'

Character Subtraction (Char - Char)

Subtracts the code of one character from another, returning the integer difference between them.

val nextCh = 'c' - 'a' // result will be: 2

Comparison Operators

Characters can be compared based on their underlying Unicode/ASCII code values using standard comparison operators (==, !=, <, >, <=, >=).

val a = 'a'
val b = 'b'

val isEqual = (a == b)       // result will be: false
val isNotEqual = (a != b)    // result will be: true
val isLess = (a < b)         // result will be: true
val isGreater = (a > b)      // result will be: false
val isLessOrEqual = (a <= b)    // result will be: true
val isGreaterOrEqual = (a >= b) // result will be: false

For Loop

A "For Loop" iterates over a sequence (like a list, array, or range) and executes a block of code for each element in that sequence. It's ideal for tasks where the number of iterations is known or defined.

1) Traditional For Loop

C-like For Loop

Iterates using a classic three-expression syntax: initialization, condition, and increment/decrement.

for(var i = 0; i < 2; i++) {
        println("c-like for: i = $i")
}

// result will be:
c-like for: i = 0
c-like for: i = 1

2) Range-Based Loops

Closed Range Loop (Inclusive)

Iterates through a range of numbers from the start value up to and including the end value.

for(i in 0 .. 2) {
        println("for in: i = $i")
}

// result will be:
for in: i = 0
for in: i = 1
for in: i = 2

Half-Open Range Loop (Exclusive)

Iterates through a range of numbers from the start value up to, but excluding, the end value.

for(i in 0 until 2) {
        println("for in: i = $i")
}
// result will be:
for in: i = 0
for in: i = 1

2) Collection Iteration

Array Iteration

Iterates directly over each element inside a given array. 

val a = [1, 2]
for(i in a) {
        println("for iterate: i = $i")
}

// result will be:

for iterate: i = 1

for iterate: i = 2

String Iteration

Iterates through each individual character of a string sequence.

for(i in "ab") {
        println("string iterate: i = $i")
}

// result will be:

string iterate: i = a

string iterate: i = b

If-Else expression

An if-else expression in Raptorian is a control flow statement that allows for conditional execution of code segments. It works as follows: the 'if' part tests a condition – if this condition is true, the code block under 'if' executes. If the condition is false, the code block under 'else' executes instead. This structure enables programs to make decisions and perform different actions based on different inputs or states. It's a fundamental concept in programming, used for branching the flow of execution in a program.

val a = 12
if(a > 10) print("a more that 10") else print("a lower than 10")
// a more that 10

val b = 24

val max = if(a > b) a else if (a < b) b else a * b
println(max) // 24

if(a == 12 && b / 2 == 12) {
        println("true: a is 12 and b / 2 is 12")
} else {
        println("false: a = $a, b = $b")
}
// true: a is 12 and b / 2 is 12

When expression

The "when" expression in Raptorian functions as a control structure for handling multiple conditional paths. It's similar to a switch-case statement in other languages. The expression evaluates a variable or condition, and based on the value, executes the corresponding code block defined under that case. This allows for cleaner, more organized code when dealing with multiple potential states or conditions, improving readability and efficiency in decision-making processes within the program. It's an effective tool for reducing complex if-else chains, especially useful in scenarios with many discrete cases to consider.

fun check(a: Int) = when(a) {
   11 -> print("a == 11")
   12 -> print("a == 12")
   else -> {
       print("a is neither 11 or 12")
   }
}

fun main() {
        var a = 42
        check(a) // a is neither 11 or 12

        a = 12
        check(a) // a == 12

        var a = 11
        check(a) // a == 11

        val b = when(a) {
                12 -> a + 2
                11 -> a * 2
                else -> a / 2
        }
        
        print(b) // 22
}

3. Description of the Parser:

Title: "Raptorian Parser"

Description:

The Raptorian parser converts a series of tokens generated by a lexer into an abstract syntax tree (AST), allowing a further compiler or interpreter to efficiently process and execute the code. It recognizes language structures such as classes, functions and expressions and ensures that the program tree is correctly constructed.

Example:

{
 "Atom": {
   "name": "Atom",
   "methods": {
     "addNumberToB": {
       "name": "addNumberToB",
       "parameters": [
         {
           "name": "a",
           "type": "Int",
           "default_value": {
             "type": "LiteralNode",

class Atom(val b: Int = 0) {
        fun addNumberToB(a: Int = 0) {
                println("addNumberToB: a = " + a + ", b = " + b + ", a + b = ${ a + b }")
        }
}


fun main() {
        val a = Atom(22)
        a.addNumberToB(1111)

      // addNumberToB: a = 1111, b = 22, a + b = 1133
}

Description:

The example demonstrates the definition of the Atom class with the addNumberToB method. This code illustrates the principles of object-oriented programming in Raptorian, as well as the use of typing and class functions. The addNumberToB method prints the values of its parameters and the sum of a and b, demonstrating how it works with strings and variables.

Operators and special symbols

Raptorian is compatible with these operators and unique symbols:

  •  + ,  - ,  * ,  / ,  %  - Operators used in mathematics.

  •  =  - Operator for assigning values.

  •  += ,  -= ,  *= ,  /= ,  %=  - Enhanced value assignment operators.

  •  ++ ,  --  - Increment and decrement operators.

  •  && ,  || ,  !  - Logical operators for 'and', 'or', 'not' functions.

  •  == ,  !=  - Operators for determining equality.

  •  < ,  > ,  <= ,  >=  - Operators for making comparisons.

  •  [ ,  ]  - Operators for declaring arrays and accessing their indices.

  •  ..  - Establishing a sequence of values.

  •  :  - Distinguishes a name from its type in a statement.

  •  ->  - Divides the condition and the main part in a branch of a 'when' expression.

  •  ;  - Differentiates various statements placed on a single line.

  •  $  - Indicates a variable or expression within a string interpolation.

Keywords

  •  class / сынып  - Used to declare a new class.

  •  fun / әдіс  - Used to declare a function.

  •  val / мән  - Declares a read-only (immutable) variable.

  •  var / айны  - Declares a mutable variable.

  •  if / егер  - Introduces a conditional expression.

  •  else / басқаша  - Specifies an alternative sequence of statements.

  •  while / әзірге  - Starts a loop that continues as long as a condition is true.

  •  when / қашан  - Used for branching the program flow based on conditions.

  •  for / үшін  - Starts a loop that iterates over a range or collection.

  •  return / қайтар  - Exits from a function and returns a value.

  •  break / тоқтат  - Terminates the nearest enclosing loop.

  •  continue / жалғастыр  - Skips the current iteration of the nearest loop.

  •  try / байқа  - Starts a block of code that will be tested for errors.

  •  catch / ұстау  - Specifies a block of code to handle any errors.

  •  finally / ақыры  - Specifies a block of code that will always be executed.

  •  in / ішінде  - Used to check if an object is in a collection, or in a for-loop.

  •  is / болса  - Used to check the type of an object.

  •  false / жалған  - Represents the boolean value false.

  •  true / шын  - Represents the boolean value true.

  •  null / жоқ  - Represents a null reference.

  •  public / жалпы  - An access modifier, indicates that something is visible to all.

  •  private / жеке  - An access modifier, indicates that something is visible only within its own class.

  •  override / басыпөту  - Indicates that a member function overrides a function from a superclass.

  •  until / дейін  - Used in loops to specify the upper limit of a range.

  •  throw / лақтыр  - Used to throw an exception.

  •  enum / тізім  - Used to declare an enumeration.

  •  constructor / құрылысшы  - Used to declare a constructor of a class.

  •  interface / байланыс  - Used to declare an interface.

  •  abstract / жоба  - Used to declare an abstract class or an abstract member.

  •  import / енгіз  - Used to import packages, classes, or other entities.
